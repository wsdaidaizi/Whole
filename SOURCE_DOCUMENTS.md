## What is included in GitHub

- core research question and design for each project;
- main model specification and controls;
- principal coefficients, sample sizes, and inference rules;
- contextual-moderation conclusions;
- interpretation boundaries;
- a cross-national synthesis.

## What is deliberately not included

- raw or row-level survey data;
- restricted or licensed microdata;
- exact household or DHS coordinates;
- local computer paths;
- stale or superseded outputs;
- unverified exploratory estimates;
- the Netherlands extension, for which no source-verified final table was supplied.

The original DOCX files remain the authoritative formatted table documents. The Markdown files in this repository are compact, public-facing core summaries rather than replacements for the complete audit packages.
